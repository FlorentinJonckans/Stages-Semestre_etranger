Les bourses scolaires

Pour les étudiants en difficultés financières, l'Etat vous finance via des bourses vos études. Pour cela, il existe plusieures formes de bourses comme les bourses du Crous : l’allocation Erasmus ou encore l’aide à la mobilité internationale.
Le montant dépend ensuite de plusieurs facteurs : situation financière, année d’étude, projet et destination.

Le montant de la bourse varie selon la destination et le type de mobilité :

  • Autriche, Danemark, Finlande, Irlande, Italie, Lichtenstein, Norvège, Suède, Royaume-Uni :
    200-300€/mois pour les études ; 350-450 €/mois pour les stages

  • Belgique, Croatie, République Tchèque, Chypre, Allemagne, Grèce, Islande, Luxembourg, Pays-Bas, Portugal, Slovénie, Espagne, Turquie :
    150-250 €/mois pour les études ; 300-400€/mois pour les stages

  • Bulgarie, Estonie, Hongrie, Lettonie, Lituanie, Malte, Pologne, Roumanie, Slovaquie, Ancienne République Yougoslave de Macédoine :
    150 à 250 € pour les études ; 300-400 €/mois pour les stages

Cursus intégré

Certaines écoles proposent à leurs étudiants d'effectuer une année d'étude à l'étranger financer par celles-ci dans le but de former l'étudiant à la vie active et ainsi d'aquérir de nouvelles compétences.
